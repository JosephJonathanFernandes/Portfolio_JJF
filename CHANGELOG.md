## Contributing to Changelog

When contributing to this repository:
1. Update the [Unreleased] section with your changes
2. Follow the format: `- Description of change (#PR)`
3. Use appropriate change types (Added, Changed, Fixed, etc.)
4. Move changes to a version section when releasing

### Example
```markdown
## [1.1.0] - 2024-02-01

### Added
- Dark mode toggle feature (#123)
- Improved accessibility support (#124)

### Fixed
- Mobile navigation overflow issue (#125)
```
